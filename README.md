# Conversando com Voz – ChatGPT 🎙️🤖

## 📌 Descrição
Este projeto integra o **ChatGPT (OpenAI API)** com o **Google Text-to-Speech (gTTS)** para criar um chatbot interativo que responde em voz.  
Você digita uma pergunta, o ChatGPT gera a resposta e o programa converte em áudio para ser reproduzido automaticamente.

---

## 🚀 Tecnologias utilizadas
- **Python 3.10+**
- **OpenAI API**
- **gTTS (Google Text-to-Speech)**
- **Git + GitHub**
